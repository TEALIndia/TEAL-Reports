# TEAL - Research Reports

## Installation

1. Install latest version of python version 3 from the [official site](www.python.org) for your OS.
2. Install `questionary` and `pymupdf` python modules with

```sh
$ pip install questionary pymupdf fitz
```

## Next Process

1. Run report.py with

**Windows**

```sh
python report.py
```

**Linux/MacOS**

```sh
python3 report.py
```

2. In your terminal you would get json data printed.
3. Copy the json data from your terminal.
4. Find the first item of your document type (Sales & Lease, Mortgage etc.) in `catalogue_new.json`.
5. Paste the copied data before that.
